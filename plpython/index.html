Python w Polsce

PySilesia jest grupą lokalną działającą na Górnym Śląsku, nie jest to jednak jedyna tego typu inicjatywa w Polsce. Na tej stronie prezentujemy zestaw godnych uwagi inicjatyw okołopythonowych:

  • Polski portal Pythonowy - Polish Python Coders Group
  • PyWaw - Warszawska Grupa Użytkowników Pythona
  • Pykonik - Krakowska społeczność Pythonistów
  • django.pl - Django w Polsce
  • PyCon PL - największa polska konferencja poświęcona językowi Python
